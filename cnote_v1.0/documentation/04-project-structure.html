Project Structure

Once you have downloaded the main file and unzipped it, you will find the following folders structure:

  • dist
    • assets
      • fonts
      • images
      • vendor
        • css
        • js
    • cnote-main
    • ecommerce
  • documentation
  • src
    • assets
      • images
      • scss
      • vendor
    • cnote-main
      • assets
        • js
        • scss
      • views
    • ecommerce
      • assets
        • js
        • scss
      • views
  • gulpfile.js
  • package.json

dist - Folder contains all template distribution files like .html, .css, .js that have been processed (compiled / minified) from src folder using Node.js and Gulp.

src - This folder holds all template source files like html, .scss, .js that are then processed (compiled / minified) to dist folder.

package.json - File includes the list of dependencies to install from npm when running npm install .

gulpfile.js - Configuration file for Gulp library. It contains all tasks you want to perform with Gulp. Learn more about it from Working with Gulp section or official Gulp documentation..

cnote-main - files for the main template

ecommerce - files for the ecommerce demo

HTML File Structure

HTML files can be found in views/ folder. SWIG allows us to insert the content from partials into other files and compile them into dist/ folder.

You can directly copy the code from _partials and you can paste it wherever you want to add or change a component

All HTML files / partials are included in src/cnote-main/views/ or src/ecommerce/views/ folder .

There you will find the following structure:

  • cnote-main
    • _partials/ - contains main footers and headers
    • components/ - all pre-made HTML components (eg.: accordions, buttons, icons, forms, colors etc.)
    • headers/ - all headers / navigation bars variants
    • homepages/ - all homepages variants
    • onepages/ - all onepages variant
    • pages/ - 404 pages, about pages, login-register, blogs etc.
  • ecommerce
    • _partials/ - contains the header and the footer belonging to shop
    • And other HTML files (the homepage, pages for checkout order, profile, single product etc. )

Basic HTML code for navbar

If you need to add a specific background color to your navbar you can simply add a class like:
g-bg-dark, g-bg-primary, g-bg-orange to cn-navbar-wrapper
More colors here

cn- stands from cnote !

Of course you can change the color of text by adding cn-navbar-dark-color or cn-navbar-white-color depends on what background color you choose.

Need a sticky navbar ? no problem, just add navbar-scrollspy to cn-navbar-wrapper but don't forget to check the body and make sure you have:
data-spy="scroll" data-target=".navbar-scrollspy" data-offset="50"

Need to change the font type or size? please check _variables.scss at "navbar section"
