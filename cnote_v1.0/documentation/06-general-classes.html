General Classes

Bootstrap includes dozens of utilitiesâ€”classes with a single purpose to reduce the frequency of highly repetitive declarations.

Along with default Bootstrap 4 utility classes Cnote introduces its own. They all live inside cnote/src/assets/scss/general-utilities and general-components file.

The general classes are made to avoid as much as possible the repetition of some CSS code lines.

It also relieves the user to search through so many files and customize the layout of a page as desired. This is possible by adding or changing some classes directly from HMTL files.

SCSS has the ability to automatically generate some classes without writing them manually using functions and variables.

These helpers can be easily recognized because of the prefix with which they are defined .g-*

.g- stands from general

In the following lines, we will present some of the general classes that Cnote has.

There are classes for :

Margins:

.g-m-* ( all margins ) .g-ml-* ( margin-left) .g-mt-* ( margin-top) .g-mr-* ( margin-right ) .g-mb-* ( margin-bottom ) .g-ml-minus-* negative margin-left .g-mt-minus-* negative margin-top .g-mr-minus-* negative margin-right .g-mb-minus-* negative margin-bottom .g-l-* ( left ) .g-t-* ( top ) .g-r-* ( right ) .g-b-* ( bottom ) .g-l-minus-* ( negative left ) .g-t-minus-* ( negative top ) .g-r-minus-* ( negative right ) .g-b-minus-* ( negative bottom ) ( * ) can be a number from 1 to 150 pixels (1-20) - 1 by 1 (20-50) - 5 by 5 (50-150) - 10 by 10

Paddings:

.g-p-* ( all paddings ) .g-pl-* ( padding-left) .g-pt-* ( padding-top) .g-pr-* ( padding-right ) .g-pb-* ( padding-bottom ) ( * ) can be a number from 1 to 150 pixels (1-20) - 1 by 1 (20-50) - 5 by 5 (50-150) - 10 by 10

Widths and Heights:

.g-w* ( 10-100pixels ) 10 by 10 .g-h*( 10 - 800pixels ) 10 by 10 until 100, 50 by 50 until 800

Background colors:

.g-bg-* ( * ) can be any color from palette colors e.g: .g-bg-blue / .g-bg-blue-200 / .g-bg-green Need a background with opacity ? No problem. You can use: .g-bg-blue-opacity-* ( * ) can be any number from 0 to 9

Borders:

.g-border ( all borders ) .g-border-l ( border-left ) .g-border-t ( border-top ) .g-border-r ( border-right ) .g-border-b ( border-bottom ) .g-border-x ( border-left + border-right) .g-border-y ( border-top + border-bottom ) NOTE: Using the classes above you will set a transparent border of 1 pixel To add a specific color to border you'll need an additional class like:
.g-border-blue ( make all borders blue )
.g-border-l-blue ( make left border blue ) .g-border-x-blue ( make left & right borders blue )
Only base colors: blue, red, green and so on If you need you can set a border width by adding the following class:
.g-border-width-* ( * ) can be 1x-5x, this mean 1pixel to 5 pixels

Font sizes:

.g-font-size-* ( * ) can be a number from 12 to 60 pixels (1 by 1) from 12-25 (5 by 5) from 25-60

Text colors:

.g-text-* ( * ) can be any color from palette colors

You can pick responsive variants to target specific viewports, e.g:
.g-mr-md-10 for margin-right: 10px ( 0.625rem ) starting at the md breakpoint
.g-p-lg-20 for padding: 20px ( 1.25rem ) starting at the lg breakpoint
