Colors

There are 3 possible ways to chage the main colors of the template:

  • Customize colors via SCSS variables
  • Replaces classes in HMTL
  • Replace them in CSS
1. Customize colors via SCSS variables

In case you want to create your own color scheme - the fastest and painless way to do that is via SASS variables.

You need to edit /src/cnote-main/assets/scss/_variables.scss.

If you want to change main colors from the ecommerce, you need to change the _variables.scss from ecommerce/ folder.

Note: if you want to add new variables, you need to add them both in src/cnote-main/ and src/ecommerce/ folders only if they use a common component because if a component exists only in cnote-main/ (for example) it does not know what value it is attributed for ecommerce/ and then errors may occur.

Blelow is the part of SASS variables file responsible for colors:

                                

// cNote Variables
// -----------------------------------------------------------------------------------

// Grays

$white:     #ffffff;
$gray-100: #f9f9f9;
$gray-200: #f4f5f6;
$gray-300: #eeeff0;
$gray-400: #e3e4e6;
$gray-500: #b9b9b9;
$gray-600: #7d7d7d;
$gray-700: #4b4b4b;
$gray-800: #2d2d2d;
$gray-900: #212121;
$gray-1000: #121212;
$black:     #000;

// Bluegrays

$bluegray-100: #f1f8fa;
$bluegray-200: #e6eff4;
$bluegray-300: #ced7dc;
$bluegray-400: #b0b9be;
$bluegray-500: #838c91;
$bluegray-600: #5b6469;
$bluegray-700: #3d464b;
$bluegray-800: #2a3237;
$bluegray-900: #1e2428;
$bluegray-1000: #151a1e;


// cNote Theme Colors

$blue:          #2680e4;
$indigo:        #6610f2;
$purple:        #9053b7;
$pink:          #eb408d;
$red:           #eb2a2f;
$orange:        #eb602c;
$brown:         #9b6730;
$yellow:        #ffd716;
$green:         #10c76e;
$teal:          #34c3ab;
$cyan:          #13c3c0;


$primary:       $cyan;
$secondary:     $gray-400;
$success:       $green;
$info:          $cyan;
$warning:       $yellow;
$danger:        $red;
$light:         $gray-100;
$dark:          $bluegray-900;

If you want to change primary color you can change the value of the variable called $primary.

2. Replace classes in HTML

You can change backgrounds or text color by changing or adding classes in HTML (.g-bg-* for backgrounds or .g-text-* for text color).

All colors are presented at the link below:

3. Replace global colors in CSS

This is the hardest way to do things done. I don't recommend it but if you prefer vanilla CSS you can replace all color hex values inside /dist/cnote-v1.0/*/assets/css/main.css with your own using "find and replace" panel of you code editor.

(*) - cnote-main/ or ecommerce/.
