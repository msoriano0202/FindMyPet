3. More about JS structure

When it comes to working on projects, most of them are going to include some type of third-party libraries.

One way to think of the vendor directory is as a more generic form of lib.

In dist/, vendor/ folder most plugins are compiled directly from node_modules in their raw formonepa.

Whereas a library is usually third-party code, vendor might be third-party assets.

I also would like to tell you more about the js plugins from src/ folder.

There you can find :

  • cnote.js - the file where some of the components are customized
  • googleMap.js - where the map is customized
  • sreveal-options.js - there are the options with which the animated pages were made (by adding the coresponding classes in your HTML)

More js plugins can be found in gulpfile.js next to vendor-js task. They are imported from node_modules folder and compiled directly into /dist/cnote-v1.0/assets/vendor/js/.
